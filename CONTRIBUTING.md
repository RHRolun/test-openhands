# Contributing to OpenHands

First off, thank you for considering contributing to OpenHands. It's people like you that make OpenHands such a great tool.

## How Can I Contribute?

If you've noticed a bug or have a question, [search the issue tracker](https://github.com/all-hands-ai/OpenHands/issues) to see if someone else in the community has already created a ticket.
If not, feel free to [create a new issue](https://github.com/all-hands-ai/OpenHands/issues/new).

## Fork & create a branch

If you're going to fix a bug or add a feature, you'll need to fork the repository and create a new branch.

```bash
git clone https://github.com/<your-username>/OpenHands.git
git checkout -b <branch-name>
```

## Commit your changes

Once you've made your changes, you'll need to commit them. Make sure to write a good commit message.

```bash
git commit -m "feat: add new feature"
```

## Submitting a pull request

Once you've committed your changes, you'll need to push them to your fork and submit a pull request.

```bash
git push origin <branch-name>
```

After you've submitted your pull request, a team member will review it and merge it if it's a good fit.
